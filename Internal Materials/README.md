# Internal Materials

Эта папка содержит внутренние материалы, которые не входят в candidate-facing маршрут.

Назначение:
- методическая подготовка экзаменаторов и лидов;
- внутренние screening/onboarding-материалы;
- внутренние аналитические комментарии по качеству программы;
- служебные документы по улучшению структуры задания.

## Что находится внутри

- [00_Examiner_Methodical_Guide.md](00_Examiner_Methodical_Guide.md) — методическая подготовка экзаменатора
- [Methodical_Notes_Cloud_Adaptation.md](Methodical_Notes_Cloud_Adaptation.md) — внутренние методические заметки по адаптации под Jira Cloud
- [Improved/README.md](Improved/README.md) — внутренний screening/onboarding-пакет
- [candidate_run_comments.md](candidate_run_comments.md) — комментарии по прогону задания с позиции кандидата
