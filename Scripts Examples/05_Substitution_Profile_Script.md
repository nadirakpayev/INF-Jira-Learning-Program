# Substitution Profile Script

Источник:
- `Source Docs/Forte-J-ITSM25+Руководство+администратора_v1.0.docx`

## `Профиль замены`

- Тип: automation / job script
- Logical name: `Автоматическое перераспределение задач и согласований на замещающего по данным Assets`

## Назначение

Скрипт ищет сотрудников, у которых в Assets указан `Замещающий`, и автоматически:
- перераспределяет задачи в Jira;
- переводит согласования;
- либо снимает назначения там, где это требуется по логике процесса.

## Предпосылки

- В Assets есть объектный тип `Сотрудник`.
- Есть атрибуты:
  - `Jira Пользователь`
  - `Замещающий`
  - `Статус`
  - `Логин пользователя`
  - `Updated`
- В Jira используются типы задач:
  - `Исполнение ЗНИ`
  - `Тестирование ЗНИ`
  - `Запрос на обслуживание`
  - `Инцидент`
  - `Согласование`
- Используются статусы:
  - `На руководителе`
  - `В работе на 1-ой линии`
  - `В работе на 2-ой линии`
  - `В работе на 3-й линии`
- Есть переход `На замещающего`

## Найденная логика

- обрабатываются только сотрудники, обновленные сегодня;
- активность замещающего определяется по атрибуту `Статус`;
- в задачах согласования обновляется поле `Руководитель`;
- для задач и согласований меняется назначение на замещающего, если он активен.

## Ограничения

- логика жестко завязана на схему Assets;
- завязана на конкретные типы задач и статусы;
- описание найдено как функциональная спецификация, без полного исходного кода скрипта.
