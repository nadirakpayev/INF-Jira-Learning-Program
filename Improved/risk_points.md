# Риск-точки кандидата

| Зона | Причина | Кого затрагивает | Тип | Что делать в Improved |
| --- | --- | --- | --- | --- |
| Блок 1: создание cloud-site | Кандидат не понимает, как доказать, что среда реально создана им | junior, intern | ambiguity | Зафиксирован обязательный evidence по среде |
| Блок 1: Components | Компоненты могут быть созданы формально, без роли в маршрутизации | all | ambiguity | Введен отдельный критерий осмысленного использования |
| Блок 1: каталог | Кандидат может уйти в лишнее количество услуг вместо качественной структуры | junior | slowdown | Зафиксирован минимальный и конечный scope |
| Блок 1: Incident + Service Request | Можно запутаться между request type, workflow, queue и automation | junior | cognitive overload | Введен минимальный обязательный набор и staged delivery |
| Блок 1: SLA/Queues | Без точного минимума кандидат либо недоделает, либо перегрузит решение | all | ambiguity | Задан фиксированный минимум по обоим процессам |
| Блок 2: CMDB test data | Кандидат может не понять, нужно ли использовать все записи или только часть | junior | ambiguity | Зафиксировано: использовать выданный датасет, но не обязательно все строки |
| Блок 2: data model | Без явного формата сдачи часть кандидатов ограничится narrative-описанием | all | hidden prerequisite | Обязательны data_model, schema.sql, seed.sql, validation_queries.sql |
| Блок 2: PostgreSQL + pgAdmin | Локальная установка может стать инфраструктурным blocker'ом | junior | blocker | Требуется простой локальный сценарий без контейнеров и отдельные setup notes |
| Блок 2: Major / Problem / Change | Кандидат может попытаться строить корпоративный full-scope вместо минимальной связки | strong junior | slowdown | Зафиксирована упрощенная связка вместо полного процесса |
| Screening decision | Если критерии не разделены по блокам, ревьюер начинает компенсировать слабый блок сильным рассказом | all | unfair filter | Решение привязано только к Блоку 1 и Блоку 2 |
| Блок 3 и Блок 4 | Кандидат может считать их обязательными до решения о найме | all | ambiguity | Явно указано: выполняются только после PASS |
| Source Docs | Большой архив создает иллюзию скрытого backlog'а | all | cognitive overload | Source Docs выведены из оцениваемого контура и объявлены контекстом только для лидов |
